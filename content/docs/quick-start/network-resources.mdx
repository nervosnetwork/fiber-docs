## Network Topology

The typical testnet topology for multi-hop payments:

```
в”Ңв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”җ        в”Ңв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”җ        в”Ңв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”җ        в”Ңв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”җ
в”Ӯ nodeA в”Ӯ в”Җв”Җв”Җв”Җв”Җв–¶ в”Ӯ node1 в”Ӯ в”Җв”Җв”Җв”Җв”Җв–¶ в”Ӯ node2 в”Ӯ в”Җв”Җв”Җв”Җв”Җв–¶ в”Ӯ nodeB в”Ӯ
в”Ӯ:8227  в”Ӯ        в”Ӯpublic в”Ӯ        в”Ӯpublic в”Ӯ        в”Ӯ:8237  в”Ӯ
в””в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”ҳ        в””в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”ҳ        в””в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”ҳ        в””в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”ҳ
  sender        relay node 1     relay node 2      receiver
```

Local nodes (nodeA, nodeB) do not need to expose a public address вҖ” they connect through the public relay nodes.

## Channel Capacity

When opening a channel, each side must reserve **99 CKB** (98 CKB for commitment lock + 1 CKB for shutdown transaction fee). This reserved amount is not available for off-chain payments.

**Example**: If you fund 499 CKB and the public node contributes 250 CKB:
- Your available balance: 499 - 99 = **400 CKB**
- Public node available: 250 - 99 = **151 CKB**
